Simple text-game quest engine for PHP (write on PHP 5.3.5).

starts from console with command:
$ php index.php

to exit game type:
exitgame


For now works only commands:

north (n)
south (s)
west (w)
east (e)

look - show description of current player location
take <item_id> - take <item_id> and put it into player inventory
drop <item_id> - drop <item_id> from inventory.

exitgame
