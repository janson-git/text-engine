text-engine
===========

some homework for text-game engine :) just for fun

Start:

  $ php index.php

Expected commands format:

  > &lt;command&gt; [&lt;argument&gt;]

For example:

  > look

  > take test_box

  > drop all

  > inventory

  > go north
  

Allowed some aliases for commands:

_n_ - go north

_s_ - go south

_w_ - go west

_e_ - go east

_i_ - inventory

_l_ - look
